Usando fontes externas

Usamos o @font-face{} quando temos a fonte baixada

Tipos de format()

format(): tem os seguintes tipos:

  • 'opentype' para otf
  • 'truetype' para ttf
  • embedded-opentype(Apple Advanced Typography)
  • truetype-aat
  • svg
