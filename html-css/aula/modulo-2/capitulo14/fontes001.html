Trabalhando com Fontes

Nem todo computador vai entender as fontes, basicabente tem que colocar fontes que contem nos dispositivos em geral, oque geralmente se faz é usar mais de uma combinação, isso é chamado de safe combinations.

Sempre termine sua 'safe combination' com uma fonte generica

Fontes Genericas

Não é uma fonte voce vai meio dizer pro navegador que quer uma fonte sem serifa (sans-serif), todos os dispositivos vao ler, pois todo dispositivo tem alguma fonte sem serifa, isso vai resultar numa escolha de fonte sem serifa aleatoria conforme os dispositivos entendem.

O mesmo vale para as fontes serifadas (serif), fontes mono-espaçadas (monoespace)

Sempre termine sua declaração segura com um nome generico
