Alinhamento de textos com CSS

Tipos de alinhamentos

Nessa aula veremos alguns tipos de alinhamentos. Atualmente temos o alinhamento para direita, esquerda e centralizada. E tambem tem o alinhamento justificado, faz com que fique alinhado tanto na esquerda como na direita. Tambem temos a declaração - text-indent(identação):30px;
