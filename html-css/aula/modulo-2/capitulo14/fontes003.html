Shorthand font

Chamada 'mãozinha' tem uma sequencia para o font: style weight size family; só usar o 'font:' e so botar o valor seguindo essa mesma ordem. font: italic bold 2m 'Work Sans', sans-serif;

Não confundir essas palavras: weight = peso; widht = largura; height = altura
