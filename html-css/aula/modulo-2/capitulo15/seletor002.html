As diferenças entre id e class

id:

Um id vai identificar um elemento unico dentro da pagina atual. Como fizemos com o primeiro h1 dessa pagina, e podemos modifica-lo de uma forma diferente dos demais h1. Em css usamos # e o nome do id. Id é unico, so 1 por pagina

class:

Já o class vai identificar uma classe à qual um ou mais elementos perteçam, nos que escolhemos o nome da classe, podendo compartilhar caracteristicas em comum a todos os que façam parte desse grupo. Em CSS usamos um .ponto antes do nome da classe. Class pode ser usado multiplas vezes

E podemos usar o id com o class
