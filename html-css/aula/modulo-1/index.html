Marca Texto

Para marcar um texto, como fiz agora, basta escrever qualquer coisa dentro da tag <mark>Aqui vai o conteudo</mark>

Para deixar um texto menor, basta escrever algo dentro da tag <small>Aqui vai o conteudo</small>

Links

Para criar um hyperlink, devemos criar ancoras atraves da tag <a>. O principal atributo dessa tag é o href, dentro desse atributo, usamos uma URL completa para o site. HREFLANG='pt-br'. TARGET = _blank, _self, _top, _parent e caso esteja usando frames usar o name do frame

Esse link é seu ou dos outros? Existe um recurso bem interessante para links que é indicar qual é a natureza do destino usando o atributo REL. Esse atributo aceita varios valores, entre eles:

Next
Indica que o link é para a proxima parte do documento atual.
Prev
Indica que o link é para a parte anterior do documento atual.
Author
Indica que é um link para o site do autor do documento atual.
External
Indica que é um link para outro site que nao faz parte do site atual.
Nofollow
Indica que é um link para um site não endosado, como um link pago.
